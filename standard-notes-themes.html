Standard Notes Themes

This page contains a collection of my Standard Notes themes.

All of my themes are published under the GPL-3.0 License.

Standard Blue

A light blue theme for Standard Notes.

To install, go to Preferences > Plugins > Install Custom Plugin

Paste the following link in the text box:

https://listed.to/p/gWG2LYgCPH

Alternatively, this theme is now included in the official Standard Notes Community Plugins repository.

GitHub: PASSK3YS/standard-blue
